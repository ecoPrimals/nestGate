## 🚀 **QUICK COMMANDS**

### **Development**
```bash
# Build project
cargo build --workspace

# Run all tests
cargo test --workspace

# Check code quality
cargo clippy --workspace
cargo fmt --all --check

# Generate documentation
cargo doc --workspace --no-deps --open
```

### **Deployment**
```bash
# Build release
cargo build --release

# Run production server
./target/release/nestgate-api-server

# Health check
curl http://localhost:8080/health
```

### **Docker**
```bash
# Build image
docker build -t nestgate:latest .

# Run container
docker run -p 8080:8080 nestgate:latest

# Docker Compose
docker-compose up -d
```
