## 🚀 **Quick Start**

### **Build & Test**

```bash
# Build
cargo build --all-targets --all-features

# Test
cargo test --workspace

# Lint
cargo clippy --all-targets --all-features -- -D warnings

# Format
cargo fmt --all
```

### **Run NestGate**

```bash
# Start server (environment-driven)
export NESTGATE_HOST="0.0.0.0"
export NESTGATE_PORT="8080"
cargo run -- serve

# Health check
cargo run -- health

# Version info
cargo run -- version
```
